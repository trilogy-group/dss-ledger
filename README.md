# dss-ledger — archived

**Intake moved 2026-07-17.** This repo no longer collects feedback. It is kept as an archived record of the pre-migration ledger.

Feedback now files through the skill front doors:

    POST https://data-source-skills.vercel.app/{source}/feedback

where `{source}` is one of `rebl3`, `red2`, `aerie`, `redshift`.

## Where issues live now

Open items from this ledger were transferred to each skill's project repo:

- **rebl3** issues → [trilogy-group/rebl3](https://github.com/trilogy-group/rebl3/issues?q=is:open+label:skill-feedback)
- **red2 / aerie / redshift / cross-system** issues → [trilogy-group/dss](https://github.com/trilogy-group/dss/issues?q=is:open+label:skill-feedback)

All previously-open issues were migrated on 2026-07-17; nothing was deleted. This repository is archived (read-only).
